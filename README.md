# RE.USE - Frontend

Frontend do brechó fictício **Re.use**, desenvolvido com **React**, **Tailwind CSS** e **Vite**.  
O site consome a API do [Re.use Backend](https://reuse-api.onrender.com) e exibe os produtos de forma rápida e moderna.


> verifique o site em: https://reusebrecho.netlify.app/

## Tecnologias Utilizadas

- [React](https://react.dev/) — Biblioteca JavaScript para construção de interfaces
- [Vite](https://vitejs.dev/) — Ferramenta de build rápida e moderna
- [Tailwind CSS](https://tailwindcss.com/) — Framework utilitário para estilização
- [React Router](https://reactrouter.com/) — Navegação entre páginas
- [Netlify](https://www.netlify.com/) — Hospedagem do frontend

---

## Funcionalidades Principais

✅ Listagem de produtos por categoria  
✅ Exibição de detalhes de cada produto  
✅ Consumo dinâmico da API do backend (Node + MongoDB)  
✅ Layout totalmente responsivo  
✅ Design minimalista e rápido com Tailwind  
✅ Carrinho de compras simples (com estado local)  

---

## Instalação e Execução Local

### 1 - Clonar e abrir o repositório
```bash
git clone https://github.com/Juh-MM/Re.use
cd Re.use
```
### 2 - Rodar o código

```bash
npm install
npm run dev
```
> o site estará rodando em: http://localhost:5173
